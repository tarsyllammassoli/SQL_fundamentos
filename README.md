# Exercícios de SQL

Repositório dedicado à prática de consultas SQL, modelagem básica de dados e manipulação de informações em bancos relacionais.

---

## Objetivo

Desenvolver domínio em SQL, incluindo:

* Consultas básicas e avançadas
* Filtragem e ordenação de dados
* Funções de agregação
* Agrupamentos e filtros com `GROUP BY` e `HAVING`
* Junções entre tabelas (`JOIN`)
* Manipulação de dados (DML)

---

## Conteúdo

---

## Tópicos Praticados

* `SELECT`, `FROM`, `WHERE`
* `ORDER BY`, `DISTINCT`
* `COUNT`, `SUM`, `AVG`, `MIN`, `MAX`
* `GROUP BY`, `HAVING`
* `INNER JOIN`, `LEFT JOIN`
* `INSERT`, `UPDATE`, `DELETE`
* Criação e alteração de tabelas (`CREATE`, `ALTER`)

---

## Obs.:

Este repositório tem finalidade educacional e documenta a evolução no aprendizado de bancos de dados e SQL.
